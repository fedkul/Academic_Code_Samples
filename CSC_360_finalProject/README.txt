To run the project simply run the index.html page from the project folder in whatever browser you prefer.

For Github viewers, This is the last project of a Javascript class.  In this project I attempted to design a webpage that parses other webpages for content and then displays them in an organized manner.  The idea behind it is to declutter webpages with a lot of ads that would otherwise hide the content you orignially went for (such as an article or image gallery).
